Detalhes do Projeto CargoBot

O CargoBot é um robô seguidor de linha desenvolvido com Arduino, sensores infravermelhos e motores DC.

Componentes Utilizados

  • Arduino Uno R3 / Nano
  • 2x Sensores Infravermelhos (IR) Tcrt5000
  • 2x Motores DC + Ponte H (L298N)
  • 2x Rodas + Roda boba
  • Estrutura leve (papelão, acrílico, MDF)
  • Pilha
  • 1x Botão On/Off
  • Cabos Jumper

Funcionamento

O robô segue uma linha preta em fundo branco, utilizando sensores infravermelhos para detectar o caminho. O sistema de controle ajusta os motores para manter o robô na trajetória correta.
